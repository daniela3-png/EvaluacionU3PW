# Gestor de Tareas Universitarias (Task Planner)

## Objetivo del Proyecto
Aplicación Frontend interactiva diseñada para la organización y planificación de entregas académicas. Permite a los estudiantes registrar asignaciones con validaciones rigurosas en tiempo real y asegurar la persistencia de los datos de manera local.

## Requerimientos Cumplidos (Rúbrica Unidad III)
- **Estructura y Maqueta**: Separación estricta de responsabilidades (HTML5, CSS3, JS Vanilla). Diseño modular y responsive adaptativo a dispositivos móviles.
- **Interacción con el DOM y Eventos**: Implementación de delegación de eventos (`click`, `submit`, `input`). Inserción y remoción dinámica de elementos sin recargar la página.
- **Formulario y Validaciones**: Formulario de 4 campos con 5 reglas de validación personalizadas (campos requeridos, longitud mínima, control de fechas futuras y confirmación de coincidencia).
- **Persistencia**: Implementación completa del flujo CRUD local a través de la API `LocalStorage`.
- **Calidad de código**: Testing realizado exitosamente en Google Chrome y Mozilla Firefox sin registrar advertencias ni errores en la Consola de Desarrollador (DevTools).
